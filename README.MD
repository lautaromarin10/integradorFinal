# Proyecto Integrador Final Programación II

Este proyecto implementa un sistema de gestión para Mascotas y sus respectivos Microchips, con una arquitectura de capas.

## Descripción general

El sistema permite registrar Mascotas y sus Microchips, actualizarlos, eliminarlos y hacer consultas

## Estructura

Todo el proyecto esta estructurado usando arquitectura en capas: Entidades, DAOs, Services y capa de presentación

### Detalle de la estructura

- src/Entities: Entidadaes principales
- src/Dao: Data Access Object
- src/Services: Lógica de negocio y validaciones
- src/Main: Menú y entrada al programa
- src/Config: Conexión a la Base de Datos.

## Elementos adicionales

- UML: Diagrama UML utilizado para realizar el proyecto.
- SQL/instrucciones.sql: Creación de la base de dato y tablas principales
- SQL/datos_pruebas.sql: Carga de animales y microchips para realizar pruebas
